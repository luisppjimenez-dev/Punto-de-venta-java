Reporte de Avance

Proyecto: Sistema de Punto de Venta
Lenguaje: Java
Entorno de desarrollo: IntelliJ IDEA con Git y GitHub parra poder tabajar en equipo
 
en pocas palabras se hizo:
1. Creación de cuenta en GitHub
Se creó una cuenta en la plataforma GitHub, la cual permite almacenar proyectos de programación
 en repositorios y facilita el trabajo colaborativo entre los niembros del equipo.

2. Instalación y verificación de Git
Se instaló el sistema de control de versiones Git en la computadora.

3. Configuración inicial de Git
Se configuraron los datos de usuario necesarios para registrar los cambios en el repositorio. 
Esto permite identificar quién realiza cada modificación en el proyecto.

4. Creación del repositorio en GitHub
Se creó un repositorio en GitHub con el nombre del proyecto. 
Durante su creación se configuraron las siguientes opciones:
Se agregó un archivo .gitignore configurado para proyectos Java.
No se agregó licencia al repositorio.
Se añadió un archivo README.md para describir el proyecto.
El repositorio quedó listo para almacenar el código del proyecto.

5. Clonación del repositorio
El repositorio fue clonado desde GitHub hacia la computadora local utilizando IntelliJ IDEA. 
Esto permite trabajar en el proyecto de manera local y sincronizar los cambios con el repositorio en línea.

6. Apertura del proyecto en IntelliJ IDEA
El repositorio clonado se abrió en el entorno de desarrollo IntelliJ IDEA. Dentro del proyecto se
 comenzó a preparar la estructura para el desarrollo del programa Java.

7. Preparación del proyecto para desarrollo colaborativo
El repositorio fue preparado para permitir que varios integrantes del equipo puedan trabajar en el 
mismo proyecto mediante Git y GitHub. Cada integrante podrá descargar el proyecto, realizar cambios 
y subirlos al repositorio central.
